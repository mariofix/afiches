Most popular Flask packages

Flask-SQLAlchemy

Adds SQLAlchemy ORM support to your Flask application. The most widely used database integration for Flask, supporting all major relational databases including SQLite, PostgreSQL, MySQL, and MariaDB.

Database & ORM · ⭐ 4,100 stars · 3.8M downloads/mo · PyPI · GitHub

Flask-RESTful

An extension for Flask that adds support for quickly building REST APIs. It encourages best practices with minimal setup, using class-based views with automatic request parsing and HTTP exception handling.

REST / APIs · ⭐ 6,700 stars · 1.4M downloads/mo · PyPI · GitHub

Flask-Login

Provides user session management for Flask. Handles the common tasks of logging in, logging out, and remembering users' sessions over extended periods of time without any specific database requirements.

Authentication · ⭐ 3,524 stars · 2.1M downloads/mo · PyPI · GitHub

Flask-WTF

Simple integration of Flask and WTForms, including CSRF protection. Makes it straightforward to work with HTML forms, including file uploads, with automatic CSRF token generation and validation.

Forms · ⭐ 1,402 stars · 2.6M downloads/mo · PyPI · GitHub

Flask-CORS

A Flask extension for handling Cross-Origin Resource Sharing (CORS), making cross-origin AJAX requests possible. Supports all CORS features including preflight requests, credentials, and multiple origins.

Security · ⭐ 871 stars · 4.2M downloads/mo · PyPI · GitHub
